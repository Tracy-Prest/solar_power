SOLAR POWER PERFORMANCE ANALYSIS

As solar energy adoption grows worldwide, understanding how well a photovoltaic (PV) system performs is essential for:

- Identifying system inefficiencies and faults

- Measuring environmental impacts (temperature, humidity, etc.)

- Improving forecasting accuracy

- Maximizing energy yield and cost savings

- Supporting research in renewable energy and smart grid optimization

This project provides a structured technical analysis of solar power data to help improve system insights and performance.

This repository contains an analysis of solar power generation data collected every 5 minutes from October 1 to December 25, 2022.
The analysis explores:

Power output trends

Environmental influence (temperature & humidity)

Daily averages and performance patterns

Actual vs predicted power 

Interactive Power BI dashboard insights
